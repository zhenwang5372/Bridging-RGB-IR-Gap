## 2. 前置条件

### 2.1 硬件要求
- GPU: NVIDIA GPU (建议显存 >= 8GB)
- 支持的 GPU 架构: Volta (sm_70), Turing (sm_75), Ampere (sm_80/86), Ada/Hopper (sm_90+)

### 2.2 系统要求
- 操作系统: Ubuntu 18.04/20.04/22.04
- GCC: 7.0+ (推荐 11.x)
- CUDA Toolkit: 与 PyTorch 版本匹配

## 3. 安装步骤

### 步骤 1: 创建 Conda 环境

```bash
# 创建新的 conda 环境
conda create -n yoloworld python=3.10 -y
conda activate yoloworld
```

### 步骤 2: 安装 PyTorch

根据你的 CUDA 版本选择合适的 PyTorch 安装命令：

**方案 A: CUDA 12.x (推荐，与当前项目一致)**
```bash
# CUDA 12.1/12.4+
pip install torch torchvision torchaudio --index-url https://download.pytorch.org/whl/cu121
```

**方案 B: CUDA 11.8**
```bash
pip install torch torchvision torchaudio --index-url https://download.pytorch.org/whl/cu118
```

**方案 C: CUDA 11.3 (官方推荐的原始版本)**
```bash
pip install torch==1.11.0+cu113 torchvision==0.12.0+cu113 torchaudio==0.11.0 --extra-index-url https://download.pytorch.org/whl/cu113
```

### 步骤 3: 安装 MMCV (关键步骤!)

**⚠️ 重要: MMCV 必须与 PyTorch 和 CUDA 版本匹配!**

**方法 1: 使用 openmim 安装 (推荐)**
```bash
pip install openmim
mim install mmcv==2.0.0
```

**方法 2: 使用 pip 从预编译 wheel 安装**
```bash
# CUDA 12.1, PyTorch 2.1+
pip install mmcv==2.1.0 -f https://download.openmmlab.com/mmcv/dist/cu121/torch2.1/index.html

# CUDA 11.8, PyTorch 2.0
pip install mmcv==2.0.0 -f https://download.openmmlab.com/mmcv/dist/cu118/torch2.0/index.html

# CUDA 11.7, PyTorch 1.13
pip install mmcv==2.2.0 -f https://download.openmmlab.com/mmcv/dist/cu117/torch1.13/index.html

# CUDA 11.3, PyTorch 1.11 (官方推荐)
pip install mmcv==2.0.0 -f https://download.openmmlab.com/mmcv/dist/cu113/torch1.11/index.html
```

**方法 3: 源码编译 MMCV (如果预编译版本不可用)**
```bash
# 设置环境变量
export MMCV_WITH_OPS=1
export FORCE_CUDA=1

# 克隆并安装
git clone https://github.com/open-mmlab/mmcv.git
cd mmcv
git checkout v2.0.0
pip install -e . -v

# 编译可能需要 10-30 分钟
```

### 步骤 4: 安装 MMEngine 和 MMDetection

```bash
# MMEngine
pip install mmengine==0.10.3

# MMDetection
pip install mmdet==3.0.0
```

### 步骤 5: 安装 MMYOLO (使用项目内置版本)

```bash
# 进入项目目录
cd /path/to/YOLO-World-master_2

# 安装项目内置的 mmyolo
cd third_party/mmyolo
pip install -e .
cd ../..
```

### 步骤 6: 安装其他依赖

```bash
# 基础依赖
pip install opencv-python==4.9.0.80
pip install opencv-python-headless==4.2.0.34
pip install timm==0.6.13
pip install transformers==4.36.2
pip install albumentations
pip install numpy prettytable

# Demo 依赖 (可选)
pip install gradio==4.16.0
pip install supervision
```

### 步骤 7: 安装 YOLO-World 项目

```bash
# 回到项目根目录
cd /path/to/YOLO-World-master_2

# 安装项目 (如果有 setup.py)
pip install -e .
```

## 4. 版本兼容性说明

### 4.1 OpenMMLab 组件版本对应关系

| mmcv | mmengine | mmdet | mmyolo |
|------|----------|-------|--------|
| 2.0.0 | 0.6.0-0.10.x | 3.0.0 | 0.6.0 |
| 2.1.0 | 0.8.0+ | 3.1.0+ | 0.6.0 |
| 2.2.0 | 0.10.0+ | 3.2.0+ | 0.6.0 |

### 4.2 PyTorch 与 CUDA 版本对应关系

| PyTorch | 支持的 CUDA 版本 | 推荐的 MMCV 版本 |
|---------|-----------------|-----------------|
| 1.11.0 | 10.2, 11.3, 11.5 | 2.0.0 |
| 1.13.x | 11.6, 11.7 | 2.0.0-2.2.0 |
| 2.0.x | 11.7, 11.8 | 2.0.0-2.1.0 |
| 2.1.x | 11.8, 12.1 | 2.1.0+ |
| 2.2.x+ | 12.1, 12.4+ | 2.1.0+ |

### 4.3 MMCV 编译注意事项

1. **CUDA 版本必须一致**: 编译 MMCV 使用的 CUDA 版本必须与 PyTorch 安装时使用的 CUDA 版本一致
2. **检查 CUDA_HOME**: 确保 `CUDA_HOME` 环境变量指向正确的 CUDA 安装路径
   ```bash
   echo $CUDA_HOME
   # 应该类似: /usr/local/cuda-12.1
   ```
3. **GCC 版本**: CUDA 12.x 需要 GCC 11+，CUDA 11.x 需要 GCC 9+

## 5. 常见问题排查

### 5.1 MMCV 安装失败

**问题**: `No matching distribution found for mmcv==2.0.0`

**解决方案**:
```bash
# 1. 检查 PyTorch 和 CUDA 版本
python -c "import torch; print(torch.__version__); print(torch.version.cuda)"

# 2. 从正确的索引安装
pip install mmcv==2.0.0 -f https://download.openmmlab.com/mmcv/dist/cu<CUDA版本>/torch<TORCH版本>/index.html
```

### 5.2 模块注册错误

**问题**: `'XXX is not in the mmyolo::model registry'`

**解决方案**:
```bash
# 确保 yolo_world 模块被正确导入
# 在运行脚本前添加:
export PYTHONPATH="${PYTHONPATH}:/path/to/YOLO-World-master_2"

# 或在 Python 代码中:
import sys
sys.path.insert(0, '/path/to/YOLO-World-master_2')
```

### 5.3 CUDA 版本不匹配

**问题**: `CUDA error: no kernel image is available for execution on the device`

**解决方案**:
```bash
# 检查 GPU 架构
nvidia-smi --query-gpu=compute_cap --format=csv

# 重新编译 MMCV 以支持您的 GPU 架构
export TORCH_CUDA_ARCH_LIST="7.0 7.5 8.0 8.6 8.9 9.0"
pip install mmcv --no-cache-dir -v
```

### 5.4 transformers 版本冲突

**问题**: CLIP 模型加载失败

**解决方案**:
```bash
pip install transformers==4.36.2
```

## 6. 环境验证脚本

创建一个验证脚本来检查环境是否正确安装：

```python
#!/usr/bin/env python
# -*- coding: utf-8 -*-
"""
环境验证脚本 - 检查 YOLO-World 运行环境
保存为: check_env.py
运行: python check_env.py
"""

import sys

def check_version(module_name, expected_version=None):
    """检查模块版本"""
    try:
        module = __import__(module_name)
        version = getattr(module, '__version__', 'unknown')
        status = "✓"
        if expected_version and version != expected_version:
            status = "⚠"
        print(f"  {status} {module_name}: {version}")
        return True
    except ImportError as e:
        print(f"  ✗ {module_name}: 未安装 ({e})")
        return False

def main():
    print("=" * 60)
    print("YOLO-World 环境检查")
    print("=" * 60)
    
    all_ok = True
    
    # 1. 检查 Python 版本
    print("\n[1] Python 版本:")
    py_version = sys.version_info
    print(f"  Python {py_version.major}.{py_version.minor}.{py_version.micro}")
    if py_version.major != 3 or py_version.minor < 8:
        print("  ⚠ 建议使用 Python 3.8-3.10")
        all_ok = False
    
    # 2. 检查 PyTorch 和 CUDA
    print("\n[2] PyTorch 和 CUDA:")
    try:
        import torch
        print(f"  ✓ PyTorch: {torch.__version__}")
        print(f"  ✓ CUDA 可用: {torch.cuda.is_available()}")
        if torch.cuda.is_available():
            print(f"  ✓ CUDA 版本: {torch.version.cuda}")
            print(f"  ✓ cuDNN 版本: {torch.backends.cudnn.version()}")
            print(f"  ✓ GPU: {torch.cuda.get_device_name(0)}")
    except ImportError:
        print("  ✗ PyTorch 未安装")
        all_ok = False
    
    # 3. 检查 OpenMMLab 组件
    print("\n[3] OpenMMLab 组件:")
    components = [
        ('mmcv', '2.0.0'),
        ('mmengine', '0.10.3'),
        ('mmdet', '3.0.0'),
        ('mmyolo', '0.6.0'),
    ]
    for name, version in components:
        if not check_version(name, version):
            all_ok = False
    
    # 4. 检查 MMCV CUDA 算子
    print("\n[4] MMCV CUDA 算子:")
    try:
        from mmcv.ops import nms
        print("  ✓ MMCV CUDA 算子可用")
    except Exception as e:
        print(f"  ✗ MMCV CUDA 算子不可用: {e}")
        all_ok = False
    
    # 5. 检查其他依赖
    print("\n[5] 其他依赖:")
    deps = [
        ('cv2', None),  # opencv-python
        ('transformers', '4.36.2'),
        ('timm', '0.6.13'),
        ('numpy', None),
        ('albumentations', None),
    ]
    for name, version in deps:
        check_version(name, version)
    
    # 6. 检查 YOLO-World 模块
    print("\n[6] YOLO-World 模块:")
    try:
        import yolo_world
        print("  ✓ yolo_world 模块可导入")
    except ImportError as e:
        print(f"  ⚠ yolo_world 模块导入失败: {e}")
        print("  请确保 PYTHONPATH 包含项目根目录")
    
    # 总结
    print("\n" + "=" * 60)
    if all_ok:
        print("✓ 环境检查通过!")
    else:
        print("⚠ 存在一些问题，请根据上述信息修复")
    print("=" * 60)

if __name__ == '__main__':
    main()
```

## 快速安装命令汇总

以下是一键安装命令（假设 CUDA 12.1 环境）：

```bash
# 创建环境
conda create -n yoloworld python=3.10 -y
conda activate yoloworld

# 安装 PyTorch
pip install torch torchvision torchaudio --index-url https://download.pytorch.org/whl/cu121

# 安装 OpenMMLab 组件
pip install openmim
mim install mmcv==2.1.0
pip install mmengine==0.10.3
pip install mmdet==3.0.0

# 进入项目目录
cd /path/to/YOLO-World-master_2

# 安装项目内置的 mmyolo
cd third_party/mmyolo && pip install -e . && cd ../..

# 安装其他依赖
pip install opencv-python==4.9.0.80 timm==0.6.13 transformers==4.36.2 albumentations numpy prettytable

# 安装 demo 依赖 (可选)
pip install gradio==4.16.0 supervision

# 验证安装
python check_env.py
```
