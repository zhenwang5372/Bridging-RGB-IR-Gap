# MultiLevelRGBIRFusion 多尺度RGB-IR跨模态融合模块

## 1. 模块概述

`MultiLevelRGBIRFusion` 是一个轻量级的多尺度 RGB-IR 跨模态融合模块，在 FPN 的每个层级（P3、P4、P5）独立进行特征融合。核心机制是**利用 IR 特征生成空间注意力**来增强 RGB 特征，实现信息互补。

### 设计动机

RGB 和 IR 图像各有优势：
- **RGB**：丰富的颜色和纹理信息
- **IR**：对光照变化鲁棒，能看到热源

通过让 IR 特征指导 RGB 特征的增强，可以让模型在各种光照条件下都能获得可靠的检测性能。

---

## 2. 核心组件

### 2.1 LightweightCrossFusion（单尺度融合）

每个尺度使用一个 `LightweightCrossFusion` 模块进行融合：

#### 核心机制
1. **通道对齐**：将 IR 通道数对齐到 RGB 通道数
2. **注意力生成**：从 IR 特征生成空间注意力图
3. **注意力应用**：用注意力加权 RGB 特征
4. **跨模态融合**：拼接并通过卷积混合特征
5. **残差连接**：使用可学习的权重进行残差连接

## 4. 关键设计细节

### 4.1 通道对齐

```python
# 如果 IR 和 RGB 通道数不同，使用 1x1 卷积对齐
self.ir_align = nn.Conv2d(ir_channels, rgb_channels, kernel_size=1, bias=False) \
                if ir_channels != rgb_channels else nn.Identity()
```

**配置示例**:
- P3: IR 64 → RGB 128
- P4: IR 128 → RGB 256
- P5: IR 256 → RGB 512

### 4.2 注意力生成

```python
# 从 IR 特征生成空间注意力
mid_channels = max(rgb_channels // reduction, 8)  # reduction=4
self.attention_gen = nn.Sequential(
    nn.Conv2d(rgb_channels, mid_channels, kernel_size=1, bias=False),
    nn.BatchNorm2d(mid_channels),
    nn.ReLU(inplace=True),
    nn.Conv2d(mid_channels, rgb_channels, kernel_size=1, bias=False),
    nn.Sigmoid()  # 输出 [0, 1] 的注意力权重
)
```

**设计理念**：
- 使用瓶颈结构减少参数
- Sigmoid 确保注意力权重在 [0, 1] 范围内
- IR 的热源信息可以指导 RGB 关注重要区域

### 4.3 可学习残差权重

```python
# 初始为 0，训练中逐渐学习合适的融合强度
self.gamma = nn.Parameter(torch.zeros(1))

# 融合公式
output = x_rgb + self.gamma * fused
```

**优势**：
- 初始时 `gamma=0`，输出就是原始 RGB，保证训练稳定
- 训练过程中模型学习最优的融合权重

## 5. 关键参数配置

| 参数名 | 默认值 | 说明 |
|--------|--------|------|
| `rgb_channels` | [128, 256, 512] | RGB特征通道数 |
| `ir_channels` | [64, 128, 256] | IR特征通道数 |

## 7. 融合机制详解

### 7.1 IR 生成注意力的理由

```
     RGB Feature                    IR Feature
    ┌───────────┐                  ┌───────────┐
    │ 颜色/纹理 │                  │ 热源分布   │
    │ 细节丰富  │                  │ 目标轮廓   │
    │ 受光照影响│                  │ 光照鲁棒   │
    └─────┬─────┘                  └─────┬─────┘
          │                              │
          │                              ▼
          │                        ┌───────────┐
          │                        │  生成注意力 │
          │                        │  (哪里重要) │
          │                        └─────┬─────┘
          │                              │
          ▼                              ▼
    ┌─────────────────────────────────────────┐
    │         RGB × Attention                 │
    │    (IR告诉RGB关注哪些区域)              │
    └─────────────────────────────────────────┘
                      │
                      ▼
               增强后的RGB特征
```

### 7.2 残差连接的重要性

```python
# 无残差时
output = fused  # 完全依赖融合结果

# 有残差时
output = RGB + γ * fused  # RGB 保底 + 可选的融合增益

# γ 从 0 开始学习
# 训练初期: output ≈ RGB (稳定)
# 训练后期: output = RGB + γ * fused (充分融合)
```

## 9. 设计优势

1. **轻量高效**：使用 1x1 和 3x3 卷积，参数少
2. **注意力机制**：IR 指导 RGB 关注重要区域
3. **训练稳定**：残差连接 + γ 初始为 0
4. **多尺度独立**：每个尺度有独立的融合模块
5. **通道自适应**：自动处理 RGB/IR 通道数不同的情况
