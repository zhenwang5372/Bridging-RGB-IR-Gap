# LiteFFTIRBackbone 轻量级频域红外特征提取网络

## 1. 模块概述

`LiteFFTIRBackbone` 是一个专门为红外(IR)图像设计的轻量级特征提取骨干网络，结合了**空间域卷积**和**频率域处理**两种特征提取方式，旨在以较少的参数量（约为 YOLOv8s backbone 的 1/8）高效提取红外图像的多尺度特征。

### 设计动机

红外图像具有以下特点：
- 信息相对简单（主要是温度分布）
- 边缘和纹理信息丰富
- 适合频域分析（周期性热源、全局温度分布）

因此，我们设计了结合空间域和频域处理的轻量级网络，充分利用红外图像的特性。

---

## 2. 核心组件

### 2.1 SpectralBlock（频谱块）

SpectralBlock 是该网络的核心构建单元，包含两个并行分支：

#### 空间分支 (Spatial Branch)
- 使用 **Depthwise Separable Convolution** (MobileNet 风格)
- 捕获局部空间特征和边缘信息
- 参数高效

#### 频域分支 (Frequency Branch)
- 使用 **FFT (快速傅里叶变换)** 将特征转换到频率域
- 应用可学习的**幅度调制** (amplitude modulation) 和**相位调制** (phase modulation)
- 使用 **IFFT** 转换回空间域
- 捕获全局结构和周期性模式

### 2.2 网络架构

```
输入 IR 图像 [B, 3, H, W]
        ↓
┌───────────────────────────────────┐
│            Stem Layer             │
│  Conv2d(3→32, k=3, s=2) + BN + SiLU  │
│         输出: [B, 32, H/2, W/2]    │
└───────────────────────────────────┘
        ↓
┌───────────────────────────────────┐
│         Stage 1 → P3              │
│  SpectralBlock(32→64) + Conv(s=2) │
│         输出: [B, 64, H/8, W/8]   │
└───────────────────────────────────┘
        ↓
┌───────────────────────────────────┐
│         Stage 2 → P4              │
│  SpectralBlock(64→128) + Conv(s=2)│
│        输出: [B, 128, H/16, W/16] │
└───────────────────────────────────┘
        ↓
┌───────────────────────────────────┐
│         Stage 3 → P5              │
│ SpectralBlock(128→256) + Conv(s=2)│
│        输出: [B, 256, H/32, W/32] │
└───────────────────────────────────┘
        ↓
输出: (P3, P4, P5) 多尺度特征
```

## 3. SpectralBlock 详细处理流程

```
输入特征 X [B, C_in, H, W]
        ↓
        ├────────────────────┬────────────────────┐
        │                    │                    │
        ▼                    ▼                    │
┌───────────────┐    ┌───────────────┐           │
│  Spatial      │    │  Frequency    │           │
│  Branch       │    │  Branch       │           │
├───────────────┤    ├───────────────┤           │
│ Depthwise     │    │ 1x1 Conv      │           │
│ Conv (k=3)    │    │ (通道投影)     │           │
│      ↓        │    │      ↓        │           │
│ BN + SiLU     │    │ FFT (rfft2)   │           │
│      ↓        │    │      ↓        │           │
│ Pointwise     │    │ 幅度 & 相位    │           │
│ Conv (1x1)    │    │ 分离          │           │
│      ↓        │    │      ↓        │           │
│ BN            │    │ 可学习调制     │           │
│               │    │ amp_weight    │           │
│               │    │ phase_weight  │           │
│               │    │      ↓        │           │
│               │    │ IFFT (irfft2) │           │
└───────┬───────┘    └───────┬───────┘           │
        │                    │                    │
        └──────── Concat ────┘                    │
                   ↓                              │
           [B, C_out, H, W]                       │
                   ↓                              │
        ┌───────────────────┐                     │
        │  1x1 Conv Fusion  │                     │
        │  (特征混合)        │                     │
        │  BN + SiLU        │                     │
        └───────────────────┘                     │
                   ↓                              │
           输出特征 [B, C_out, H, W]              │
```

## 4. 频域处理详解

### 4.1 FFT 变换
```python
# 将空间特征转换到频率域
freq_domain = torch.fft.rfft2(freq_input, norm='ortho')
```

### 4.2 幅度和相位分离
```python
amplitude = torch.abs(freq_domain)   # 幅度谱 - 控制各频率成分的强度
phase = torch.angle(freq_domain)     # 相位谱 - 控制空间位置信息
```

### 4.3 可学习调制
```python
# 幅度调制：学习增强/抑制特定频率成分
amplitude_mod = amplitude * self.amp_weight

# 相位调制：学习调整空间对齐
phase_mod = phase + self.phase_weight
```

### 4.4 重建与逆变换
```python
# 重建复数频谱
freq_modulated = amplitude_mod * torch.exp(1j * phase_mod)

# 逆变换回空间域
freq_feat = torch.fft.irfft2(freq_modulated, s=freq_input.shape[-2:], norm='ortho')
```

## 5. 关键参数配置

| 参数名 | 默认值 | 说明 |
|--------|--------|------|
| `in_channels` | 3 | 输入通道数（RGB 或灰度扩展为3通道） |
| `base_channels` | 32 | 基础通道数 |
| `out_indices` | (0, 1, 2) | 输出的特征层索引 |
| `freq_ratio` | 0.5 | 频域分支占总通道的比例 |
| `frozen_stages` | -1 | 冻结的阶段数（-1表示不冻结） |

### 输出通道配置
- P3: `base_channels * 2` = 64
- P4: `base_channels * 4` = 128
- P5: `base_channels * 8` = 256

## 6. 与 RGB Backbone 的对比

| 特性 | YOLOv8s Backbone | LiteFFTIRBackbone |
|------|------------------|-------------------|
| 参数量 | 100% | ~12.5% (1/8) |
| P3 通道 | 128 | 64 |
| P4 通道 | 256 | 128 |
| P5 通道 | 512 | 256 |
| 特征提取 | 纯空间卷积 | 空间+频域混合 |
| 适用场景 | 通用图像 | 红外图像 |


## 9. 设计优势

1. **参数高效**：仅为 YOLOv8s 的 1/8 参数量
2. **双域特征**：同时捕获空间局部特征和频域全局特征
3. **可学习调制**：频域处理参数完全可学习
4. **即插即用**：输出格式与 YOLOv8 兼容，可直接用于 FPN
5. **红外优化**：针对红外图像特性设计
