# MultiLevelRGBIRFusionV3 多尺度RGB-IR跨模态融合模块（增强版）

## 1. 模块概述

`MultiLevelRGBIRFusionV3` 是 `MultiLevelRGBIRFusion` 的增强版本，提供了**多种融合策略**和**多种输出模式**的选择。相比 V1 版本只有单一的融合方式，V3 版本允许用户根据场景需求选择最适合的融合策略。

### 设计动机

不同的应用场景对 RGB-IR 融合有不同的需求：
- **白天场景**：RGB 信息更丰富，可能需要 RGB-dominant
- **夜间场景**：IR 信息更可靠，推荐 IR-dominant
- **特殊场景**：可能需要门控融合让模型自适应学习

---

## 2. 支持的融合策略

### 2.1 融合类型 (fusion_type)

| 类型 | 说明 | 优点 | 缺点 | 推荐场景 |
|------|------|------|------|---------|
| `senet` | SE Block 通道注意力 | 简单有效 | 无空间注意力 | 通用场景 |
| `scse` | 空间+通道注意力 | 更全面 | 计算量稍大 | 高精度需求 |
| `gated` | 空间门控融合 ★★★★★ | 最灵活，每像素自适应 | 需学习门控网络 | **强烈推荐** |
| `channel_gated` | 通道门控融合 | 轻量且灵活 | 无空间自适应 | 资源受限场景 |
| `none` | 无注意力，直接卷积 | 最轻量 | 无自适应能力 | 基准测试 |

### 2.2 输出类型 (output_type)

| 类型 | 公式 | 说明 | 推荐场景 |
|------|------|------|---------|
| `rgb_dominant` | `output = RGB + γ × fused` | RGB 为主体 | 白天、光照好 |
| `ir_dominant` | `output = IR + γ × fused` | IR 为主体 | **夜间、低光照** |
| `fused_only` | `output = fused` | 直接输出融合 | 实验对比 |

## 3. 核心组件详解

### 3.1 SEBlock（通道注意力）

```
输入 [B, C, H, W]
       │
       ▼
Global Avg Pool → [B, C, 1, 1]
       │
       ▼
FC(C → C/r) + ReLU
       │
       ▼
FC(C/r → C) + Sigmoid → 通道权重
       │
       ▼
输入 × 通道权重 → 输出
```

**特点**：学习哪些通道更重要，但不考虑空间位置。

### 3.2 SCSEBlock（空间+通道注意力）

```
输入 [B, C, H, W]
       │
       ├────────────────────┐
       │                    │
       ▼                    ▼
┌──────────────┐    ┌──────────────┐
│ Channel SE   │    │  Spatial SE  │
│              │    │              │
│ GAP → FC×2   │    │  Conv1×1     │
│    → Sigmoid │    │   → Sigmoid  │
│              │    │              │
│ [B,C,1,1]    │    │ [B,1,H,W]    │
└──────┬───────┘    └──────┬───────┘
       │                    │
       ▼                    ▼
   X × cse_out         X × sse_out
       │                    │
       └────── Add ─────────┘
               │
               ▼
            输出
```

**特点**：同时考虑通道和空间维度的重要性。

### 3.3 GatedFusion（门控融合）★ 推荐

```
输入: RGB [B, C, H, W], IR [B, C, H, W]
        │                    │
        └────── Concat ──────┘
                  │
                  ▼ [B, 2C, H, W]
        ┌─────────────────────┐
        │   Gate Network      │
        │   Conv3×3 → BN →ReLU│
        │   Conv1×1 → Sigmoid │
        └──────────┬──────────┘
                   │
                   ▼ gate [B, 1, H, W]
                   │
        ┌──────────┴──────────┐
        │                     │
        ▼                     ▼
   gate × IR          (1-gate) × RGB
        │                     │
        └────── Add ──────────┘
                │
                ▼
         Fused Output

公式: fused = gate × IR + (1 - gate) × RGB

gate ∈ [0, 1]:
  - gate → 1: 使用更多 IR（热源区域）
  - gate → 0: 使用更多 RGB（背景区域）
```

**特点**：
- 每个空间位置可以有不同的融合比例
- 模型自动学习哪里用 IR、哪里用 RGB
- 非常适合 RGB-IR 目标检测

### 3.4 ChannelGatedFusion（通道门控融合）

```
输入: RGB [B, C, H, W], IR [B, C, H, W]
        │                    │
        └────── Concat ──────┘
                  │
                  ▼ [B, 2C, H, W]
        ┌─────────────────────┐
        │   Channel Gate      │
        │   GAP → Conv×2      │
        │        → Sigmoid    │
        └──────────┬──────────┘
                   │
                   ▼ gate [B, C, 1, 1]
                   │
   fused = gate × IR + (1 - gate) × RGB
```

**特点**：比空间门控更轻量，但无法做空间级别的自适应。

## 4. 完整处理流程

```
输入:
  - rgb_feats: RGB特征 (P3, P4, P5)
  - ir_feats: IR特征 (P3, P4, P5)

对每个尺度独立处理:

┌────────────────────────────────────────────────────────────────┐
│                  LightweightCrossFusionV3                      │
├────────────────────────────────────────────────────────────────┤
│                                                                │
│  RGB [B, C_rgb, H, W]         IR [B, C_ir, H, W]               │
│        │                           │                           │
│        │                           ▼                           │
│        │                   ┌───────────────┐                   │
│        │                   │ Channel Align │                   │
│        │                   │ (C_ir → C_rgb)│                   │
│        │                   └───────┬───────┘                   │
│        │                           │                           │
│        │                           ▼ IR_aligned                │
│        │                           │                           │
│        │                           ▼                           │
│        │                   ┌───────────────┐                   │
│        │                   │ Attention Gen │                   │
│        │                   │ from IR       │                   │
│        │                   │ Conv→BN→ReLU  │                   │
│        │                   │ Conv→Sigmoid  │                   │
│        │                   └───────┬───────┘                   │
│        │                           │                           │
│        │                           ▼ attention_map             │
│        │                           │                           │
│        ▼                           │                           │
│  ┌─────────────────────────────────┴─────────────────────┐    │
│  │              RGB_attended = RGB × attention_map       │    │
│  └─────────────────────────────┬─────────────────────────┘    │
│                                │                               │
│                                │          IR_aligned           │
│                                │              │                │
│                                ▼              ▼                │
│  ┌─────────────────────────────────────────────────────────┐  │
│  │                Fusion (根据 fusion_type)                  │  │
│  ├─────────────────────────────────────────────────────────┤  │
│  │                                                          │  │
│  │  if fusion_type == 'gated':                             │  │
│  │      fused, gate = GatedFusion(RGB_att, IR_aligned)      │  │
│  │      fused = gate × IR + (1-gate) × RGB                  │  │
│  │                                                          │  │
│  │  elif fusion_type == 'senet':                           │  │
│  │      combined = Concat(RGB_att, IR_aligned)              │  │
│  │      combined = SEBlock(combined)                        │  │
│  │      fused = Conv(combined)                              │  │
│  │                                                          │  │
│  │  elif fusion_type == 'scse':                            │  │
│  │      combined = Concat(RGB_att, IR_aligned)              │  │
│  │      combined = SCSEBlock(combined)                      │  │
│  │      fused = Conv(combined)                              │  │
│  │                                                          │  │
│  └─────────────────────────────┬────────────────────────────┘  │
│                                │                               │
│                                ▼ fused                         │
│                                │                               │
│  ┌─────────────────────────────────────────────────────────┐  │
│  │              Output (根据 output_type)                    │  │
│  ├─────────────────────────────────────────────────────────┤  │
│  │                                                          │  │
│  │  if output_type == 'ir_dominant':                       │  │
│  │      output = IR_aligned + γ × fused   (γ 可学习)        │  │
│  │                                                          │  │
│  │  elif output_type == 'rgb_dominant':                    │  │
│  │      output = RGB + γ × fused                            │  │
│  │                                                          │  │
│  │  elif output_type == 'fused_only':                      │  │
│  │      output = fused                                      │  │
│  │                                                          │  │
│  └─────────────────────────────┬────────────────────────────┘  │
│                                │                               │
│                                ▼                               │
│                        Output [B, C_rgb, H, W]                 │
│                                                                │
└────────────────────────────────────────────────────────────────┘
```

## 5. 关键参数配置

| 参数名 | 默认值 | 说明 |
|--------|--------|------|
| `rgb_channels` | [128, 256, 512] | RGB 特征通道数 |
| `ir_channels` | [64, 128, 256] | IR 特征通道数 |
| `reduction` | 4 | 注意力模块的通道缩减比例 |
| `fusion_type` | 'gated' | 融合类型 |
| `output_type` | 'ir_dominant' | 输出类型 |

## 8. 使用示例

```python
# 配置文件中的使用方式

# 推荐配置（夜间场景）
fusion_module=dict(
    type='MultiLevelRGBIRFusionV3',
    rgb_channels=[128, 256, 512],
    ir_channels=[64, 128, 256],
    reduction=4,
    fusion_type='gated',       # 空间门控融合
    output_type='ir_dominant', # IR 为主体
)

# 白天场景配置
fusion_module=dict(
    type='MultiLevelRGBIRFusionV3',
    rgb_channels=[128, 256, 512],
    ir_channels=[64, 128, 256],
    reduction=4,
    fusion_type='gated',
    output_type='rgb_dominant', # RGB 为主体
)

# 轻量配置
fusion_module=dict(
    type='MultiLevelRGBIRFusionV3',
    rgb_channels=[128, 256, 512],
    ir_channels=[64, 128, 256],
    reduction=4,
    fusion_type='channel_gated', # 通道门控，更轻量
    output_type='fused_only',
)
```

## 9. 与 V1 版本的对比

| 特性 | V1 (MultiLevelRGBIRFusion) | V3 (MultiLevelRGBIRFusionV3) |
|------|---------------------------|------------------------------|
| 融合策略 | 固定（IR生成attention） | **可选择** (5种策略) |
| 输出模式 | 固定（RGB-dominant） | **可选择** (3种模式) |
| 门控融合 | 不支持 | **支持** (空间/通道) |
| 场景适应性 | 一般 | **强** |
| 灵活性 | 低 | **高** |

---

## 10. 设计优势

1. **策略可选**：5种融合策略适应不同场景
2. **输出灵活**：3种输出模式满足不同需求
3. **门控融合**：每个像素自适应选择 RGB/IR
4. **夜间优化**：IR-dominant 模式专为低光照设计
5. **可学习权重**：gamma 参数控制融合强度
6. **即插即用**：与 V1 接口兼容

## 11. 推荐配置

| 场景 | fusion_type | output_type |
|------|-------------|-------------|
| **LLVIP 夜间** | `gated` | `ir_dominant` |
| **FLIR 综合** | `gated` | `ir_dominant` |
| 白天为主 | `gated` | `rgb_dominant` |
| 追求轻量 | `channel_gated` | `fused_only` |
| 基准测试 | `none` | `fused_only` |
